# Chat UI Project (Gen AI Assignment)

## Project Overview

This project is a modern and responsive Chat User Interface inspired by applications like ChatGPT and Claude. The objective of this assignment is to design a clean, interactive, and user-friendly chat application using frontend technologies only.

The application allows users to send messages and receive simulated AI responses with a typing animation effect. The focus of this project is on UI/UX design, smooth interactions, and responsiveness across different devices.

---

## Features Implemented

### Chat Functionality

* Users can send messages
* AI generates mock responses
* Messages are displayed with timestamps
* Automatic scrolling to the latest message

### Typing Indicator

* Animated typing indicator to simulate AI response
* Improves user experience by showing processing state

### Input Handling

* Auto-resizing textarea
* Enter key sends message
* Shift + Enter creates a new line
* Prevents sending empty messages

### Welcome Screen

* Displays suggestion cards
* Cards are clickable and trigger predefined messages

### Sidebar

* Contains chat history
* New Chat button resets the conversation
* Sidebar toggle functionality for responsiveness

### Responsive Design

* Works across desktop, tablet, and mobile devices
* Layout adjusts based on screen size

---

## Technologies Used

* HTML5 for structure
* CSS3 for styling and animations
* JavaScript for interactivity
* Bootstrap 5 for responsive layout
* Font Awesome for icons
